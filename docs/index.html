Yoink

Capture, annotate, OCR, translate, make stickers, record video, and upload. All in one open-source tool.

Download for Windows Source Code

Region & Window Capture

Capture any region, fullscreen, active window, or scrolling content with pixel-perfect precision.

Annotation Tools

Arrows, text, shapes, blur, freehand draw, step numbers, emoji, and more, right on the capture overlay.

OCR & Translate

Extract text from any part of your screen and translate it instantly with built-in Windows OCR.

Screen Recording

Record your screen as GIF, MP4, WebM, or MKV with microphone and desktop audio support.

Stickers

Turn any capture into a sticker by removing the background, locally or with cloud providers.

Upload Anywhere

Upload to Imgur, S3, Dropbox, Google Drive, GitHub, or your own custom HTTP endpoint.
